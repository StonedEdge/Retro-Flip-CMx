# Retro-Flip-CM5
Stay tuned... dual screen flip handheld? Or something like that. Upgrades over the Retro Lite CM5 will include:

<p align="center">
  <img src="https://i.imgur.com/affNIhR.gif" width="600" />
</p>

- Analog triggers via RP2350B and hall sensors
- BQ25792 with dual power path support for VBUS input (or maybe just keep using BQ24292i)
- TPS65988D (for 2 USB-C ports) - one port supports USB 3.0, OTG and DP. Other port supports USB 2.0 and PD
- USB hub for dual USB-C ports
- Hall sensor for lid switch
- Better analog sticks (not switch sticks) - 3DS sliders. Rear back buttons for L3/R3
- Custom heatpipe plus finstack
- Button for entering maskrom mode for programming eMMC
- Two screens (OLED 1080p, 480p bottom display)
- Two 3DS XL hinges
- Blackberry 8250 finger IR touchpad
- Hotkeys
- Gyro
- RK3588 Compute Module
